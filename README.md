# CMMS Pro – Flota Frigorífica

Sistema CMMS desarrollado en Python y Streamlit para la gestión de mantenimiento
y control de repuestos de una flota de camiones con equipos frigoríficos Thermo King.

## Funcionalidades
- Gestión de Órdenes de Trabajo (Preventivo / Correctivo)
- Kardex digital de repuestos
- Consumo automático por kilometraje u horas
- KPIs de mantenimiento (Disponibilidad, MTBF, MTTR)
- Simulación de telemetría frigorífica
- Alertas automáticas de stock

## Tecnologías
- Python
- Streamlit
- Pandas
- NumPy

## Ejecución local
```bash
pip install -r requirements.txt
streamlit run app.py
